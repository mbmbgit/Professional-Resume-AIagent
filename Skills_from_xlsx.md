### 1. 人的資本データ抽出・分析パイプライン構築（海外企業PDF 800件）

| 項目 | 内容 |
| :--- | :--- |
| **期間** | 2026年3月 〜 2026年4月 |
| **役割 / 体制** | AIエンジニア・PG / 1名 |
| **使用言語** | Python 3 |
| **DB** | Mysql |
| **サーバOS** | Github Actions(Ubuntsu) |
| **FW・MW・ツール等** | Gemini 2.5 Flash, Claude Code, Docker |

**業務内容・詳細機能:**  
海外企業の人的資本レポート（アニュアルレポート等）800件から、複雑な非構造化データを高精度に抽出するAIエージェント・ワークフローを構築。
- マルチモーダルLLMの視覚能力活用:
　　従来のOCR（文字認識）を介さず、Gemini 2.5 Flashに直接PDFを「画像」として認識させる手法を採用。文字情報だけでなく、グラフの推移、チェックボックスの有無、表のレイアウト構造を視覚的に捉えたデータ抽出を実現。
- 高度なプロンプトエンジニアリング（CoT & フェイルセーフ設計）:
　　抽出精度と信頼性を極限まで高めるため、以下のロジックをプロンプトに組み込み実装。
　　* Chain of Thought (思考の連鎖): 「目次からのページ特定 → 該当箇所の視覚的スキャン → 数値特定・計算 → 結論」というステップバイステップの思考プロセスを強制し、推論ミスを大幅に削減。
　　* 人間とAIのためのフェイルセーフ設定: グラフが存在しても数値ラベルがない場合に「無理に数値を推測（ハルシネーション）しない」制約を厳格に適用。値には「グラフあり・詳細数値不明」と出力させ、理由を報告させることで、後続の人間によるチェックコストを最小化。
- トレーサビリティの徹底:
　　抽出した全データに対し、参照したページ番号と原文テキストの引用をセットで出力。コンサルタントが即座にエビデンスを確認できる仕組みを構築。

**ビジネス上の効果:**  
- データ品質の劇的向上: CoTの導入により、複雑な計算を伴う指標（一人当たりの利益率等）の抽出精度が向上。
- 検証コストの削減: AIの判断根拠（思考プロセスと参照ページ）が明示されるため、人間によるダブルチェックが効率化され、施策全体のROIが向上。
